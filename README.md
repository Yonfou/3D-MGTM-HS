# 3D Magnetic Gradient Tensor Modeling at High Susceptibility: 3D-MGTM-HS

Principal author **Fang Ouyang**,
main co-developers Longwei Chen

This is a FORTRAN program to compute the three magnetic field components and six magnetic gradient tensor components at high susceptibility.

Note: One should rewrite the subroutine (Model.f90) to create a new magnetic model. In the original version of the subroutine Model.f90, four simple models are provided:

1-a sphere
2-a spherical shell
3-a ellipsoid
4-a two dike model
